Getting Started

Nicholas Viau

January 16, 2018

R is notorious for a steep learning curve. Much of this has to do with how R was designed and how it is maintained and developed.

Râ€™s base functionality is extended with packages. An R package is a collection of functions, data, and documentation that to work with data. Package development in R is organic and anybody can create and share them. While the popularity of R is thanks to the proliferation of user-built packages, this sort of organic development means that are many ways to do the same thing, often at varying levels of efficiency, code readability, and system cost. This can also mean learning the language and doing basic tasks can be very difficult if you do not understand the language well.

While the pitfalls are avoidable, they often go unnoticed due to frustration associated with the learning process, and pitfalls soon turn into bad habits. Tidy Data

Tidy data is a term that describes a standardized approach to structuring datasets that simplify analysis and visualization. As described by Hadley Wickham (Wickham 2014), tidy data have a specific structure:

  • Each variable is a column.
  • Each observation is a row.
  • Each type of observational unit is a table.

The third the first two criterion make the third a given, so the majority of this guide will focus on the first two.
